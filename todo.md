# Doce Prazer - Cardápio Interativo Online

## Design Guidelines

### Design References
- Confeitaria artesanal premium, estilo romântico e elegante
- Identidade visual rosa/dourado da Doce Prazer

### Color Palette
- Primary: #D4708F (Rosa confeitaria)
- Secondary: #F5E6CC (Bege dourado)
- Accent: #C9A96E (Dourado)
- Dark: #3D2B1F (Marrom chocolate)
- Light: #FFF5F5 (Rosa claro)
- White: #FFFFFF
- Text: #3D2B1F (Marrom escuro)

### Typography
- Heading: 'Playfair Display', serif (elegante)
- Body: 'Poppins', sans-serif (moderno e legível)

### Key Component Styles
- Cards: Fundo branco, sombra suave, bordas arredondadas 16px
- Botões +/-: Circulares, rosa com hover dourado
- Botão "Fazer Pedido": Rosa gradiente, texto branco, grande
- WhatsApp flutuante: Verde #25D366, canto inferior direito

## Development Tasks

1. **index.html** - Estrutura completa: header com logo, hero banner, seções do cardápio por categoria, modal de resumo, modal de QR Code PIX, botão WhatsApp flutuante
2. **style.css** - Estilos completos com identidade visual rosa/dourado, responsivo com @media queries para mobile (<480px), tablet (481-768px), laptop (769-1024px) e desktop (>1024px)
3. **script.js** - Lógica do carrinho (adicionar/remover itens, calcular total), seletor de sabores, modal de resumo, geração de QR Code PIX, redirecionamento WhatsApp

## Arquivos
- index.html
- style.css
